About this Site

Hex-online.net is the best place to play Hex against your friends over the internet. Includes time control and customizable settings for how much time each player has and who goes first.

Why did I make this

Mostly as a fun programming challenge. Also, I wasn't completely satisfied with the user interface of existing Hex websites. I think hex-online improves on a lot of them. I also submitted an earlier version of this project as my final project for CS50's Web Programming with Python and JavaScript.

How to use

Simply register by choosing a user name and a password. Your opponent also needs to register and then one of you can create a game and the other can join. (You probably won't find random opponents, so it's best to arrange with a friend.)
